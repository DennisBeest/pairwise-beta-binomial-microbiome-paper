
# simulation code and data

This folder contains the code and data used run the simulations described in the paper.
Next to the data, data data folder also contains fitted coefficients for the ratios.

Note that the code takes a long time to run.

Also note that during analysis the code had to be updated frequently, 
due to alternations in underlying packages. 
It has not been updated since publication (and it is not maintained),
it is thus possible that code will not work.
