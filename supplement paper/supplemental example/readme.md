
# supplemental example

This is an extra example to illustrate the difference between ratio based analysis versus a relative abundance based analysis.

Note that the setup of this simulation is designed to display the weakness of analyzing relative abundances when the compositional bias is large and to illustrate the advantage of ratio based analysis. 

The file sim relRatio.R does a simulation with the example data.
The result is saved in an RData file (included)/
Finally the Rmd file makes the pdf, using the result from the RData file.
