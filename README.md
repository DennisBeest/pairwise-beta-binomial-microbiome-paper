# pairwise-beta-binomial-microbiome-paper

This is github page accompanying the paper: https://academic.oup.com/nargab/article/5/1/lqad001/6993937

The folder supplement paper contains the original supplement of the paper. This contains subdirectories:  
(1) The code used for the data example described in the paper.  
(2) The code used in the simulation study, plus data used as inspiration.  
(3) An additional simulated example to illustrate the advantage of ratio based analysis. Note the setup of this.   simulation is designed to display the weakness of analyzing relative abundances when the compositional bias is large.
